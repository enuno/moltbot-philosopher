## Overview

Enhance `relevance-calculator.ts` from basic 3-factor scoring (60% semantic + 25% keyword + 15% author) to a 5-factor system that incorporates trending detection, recency boosting, and quality gating via hybrid banned phrases.

---

## Architecture

### New Scoring Components

```typescript
// Factor 1: Semantic Relevance (40% → reduced from 60%)
getSemanticScore(post: Post, agent: Agent): number
  // Integrates with Noosphere for tradition matching

// Factor 2: Trending Score (NEW - 30% weight)
calculateTrendingScore(post: Post, agentType: string): number
  // Multi-layer: engagement velocity (50%) + feed trends (30%) + agent interests (20%)

// Factor 3: Keyword Match (15% → from 25%)
keywordMatch(content: string, tradition: string): number
  // Existing logic, reduced weight

// Factor 4: Author Quality (10% → from 15%)
getAuthorQuality(author): number
  // Existing logic, reduced weight

// Factor 5: Quality Gate (NEW - binary penalty)
isLowQualityComment(text: string, agentType: string): boolean
  // Returns true if comment matches GLOBAL_BANNED or AGENT_BANNED[agentType]
  // If true, multiply final score by 0.5 (50% penalty)

```

### Final Scoring Formula

```typescript
scorePost(post: Post, agent: Agent): number {
  let score =
    this.getSemanticScore(post, agent) * 0.40 +
    this.calculateTrendingScore(post, agent.type) * 0.30 +
    this.keywordMatch(post.content, agent.tradition) * 0.15 +
    this.getAuthorQuality(post.author) * 0.10;

  // Quality gate: low-effort comments get 50% penalty
  if (this.isLowQualityComment(post.content, agent.type)) {
    score *= 0.5;
  }

  return Math.min(score, 1.0);
}

```

## Component 1: Banned Phrases (Hybrid Approach)

### Global Banned List (20-25 phrases)

Universal "low-effort engagement" patterns applied to all agents:

```
"good", "good point", "interesting", "nice post", "well said",
"couldn't agree more", "this is great", "+1", "same",
"totally agree", "so true", "facts", "preach", "this x100",
"love this", "well put", "great insight", "amazing",
"thanks for sharing", "couldn't have said it better",
"exactly", "100%", "💯", "🔥", "👏"

```

### Per-Agent Banned Lists (5-7 phrases each)

Philosophy-specific anti-clichés that contradict each tradition:

**Classical**: "just be yourself", "do what feels right", "follow your heart", "listen to your gut", "trust your instincts", "good vibes"

**Existentialist**: "everything happens for a reason", "it'll all work out", "things always work themselves out", "what doesn't kill you"

**Scientist-Empiricist**: "science says...", "studies show...", "research proves...", "basically...", "obviously...", "clearly..."

**Transcendentalist**: "just manifest it", "put it in the universe", "vibes only", "spiritual bypassing"

**Beat-Generation**: "it's all meaningless anyway", "nothing matters", "the system"

**Cyberpunk-Posthumanist**: "return to nature", "humans are pure", "go off-grid", "technology bad"

**Existentialist**: "live, laugh, love", "good things come to those who wait"

**Satirist-Absurdist**: "there's a reasonable explanation", "it all makes sense", "logic will prevail"

**Joyce-Stream**: "get to the point", "be clear", "say what you mean"

### Implementation Pattern

```typescript
// banned-phrases.ts
export const GLOBAL_BANNED_PHRASES = [/* 20-25 phrases */];
export const AGENT_BANNED_PHRASES: Record<string, string[]> = {
  classical: [/* 6 phrases */],
  existentialist: [/* 6 phrases */],
  // ... 7 more agents
};

// relevance-calculator.ts
isLowQualityComment(text: string, agentType: string): boolean {
  const lower = text.toLowerCase().trim();

  if (GLOBAL_BANNED_PHRASES.some(phrase => lower.includes(phrase))) {
    return true;
  }

  const agentBanned = AGENT_BANNED_PHRASES[agentType] || [];
  if (agentBanned.some(phrase => lower.includes(phrase))) {
    return true;
  }

  return false;
}

```

## Component 2: Trending Detection (Multi-Layered)

### Layer 1: Engagement Velocity (50% of trending score)

Measures post activity momentum (comments/hour with recency boost).

```typescript
calculateVelocityScore(post: Post): number {
  const ageMs = Date.now() - post.createdAt;
  const commentsPerHour = (post.commentCount / ageMs) * 3600000;

  // Recency boost as required by P2.1
  if (ageMs < 3600000) return commentsPerHour * 1.5;  // <1h: +50% boost
  if (ageMs < 86400000) return commentsPerHour * 1.2; // <24h: +20% boost
  return commentsPerHour * 1.0;
}

```

### Layer 2: Feed Trend Matching (30% of trending score)

Checks if post content aligns with platform-wide trending topics.

```typescript
calculateFeedTrendScore(post: Post): number {
  const trendingTopics = this.getTrendingTopics(); // From state-manager
  const contentTopics = this.extractTopics(post.content);

  const matches = contentTopics.filter(t =>
    trendingTopics.find(tr => tr.topic === t)
  ).length;

  return Math.min(matches / contentTopics.length, 1.0);
}

```

### Layer 3: Agent Interest Alignment (20% of trending score)

Boosts posts that align with each agent's philosophical interests.

```typescript
calculateAgentRelevance(post: Post, agentType: string): number {
  const interests = this.getAgentInterests(agentType);
  const matchScore = this.semanticMatch(post.content, interests);
  return matchScore > 0.7 ? 1.0 : matchScore;
}

```

### Combined Trending Score

```typescript
calculateTrendingScore(post: Post, agentType: string): number {
  const velocity = this.calculateVelocityScore(post);
  const feedTrend = this.calculateFeedTrendScore(post);
  const agentRelevance = this.calculateAgentRelevance(post, agentType);

  return (velocity * 0.5) + (feedTrend * 0.3) + (agentRelevance * 0.2);
}

```
