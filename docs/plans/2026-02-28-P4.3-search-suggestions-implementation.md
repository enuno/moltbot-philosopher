# P4.3: Search Query Suggestions Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Implement trending detection batch job + suggestion ranker with dual API endpoints (/search/autocomplete, /search/related) using environment-driven weight tuning.

**Architecture:** Hourly cron job detects trending topics from query audit logs and writes to JSON state file. Suggestion ranker loads state in-memory with 5-min cache refresh, computes semantic similarity per-request, blends scores via environment weights. Two REST endpoints expose autocomplete (prefix-filtered, trending-weighted) and related-search (semantic-first).

**Tech Stack:** TypeScript (ranker + routes), Bash (trending detector), Jest (unit + integration tests), Express, Venice AI API (embeddings), Noosphere service integration.

```typescript
// services/noosphere-service/src/suggestions/ranker.ts
import { TrendingTopic, SuggestionWeights, SuggestionContext } from "./types";

/**

 * Compute cosine similarity between two embeddings.

 * Normalized to [0, 1] via (cos + 1) / 2.

 */
export function computeSemanticSimilarity(
  queryEmbedding: number[],
  topicEmbedding?: number[]
): number {
  if (!topicEmbedding || topicEmbedding.length === 0) {
    return 0;
  }

  // Cosine similarity
  const dot = queryEmbedding.reduce((acc, v, i) => acc + v * topicEmbedding[i], 0);
  const normQ = Math.sqrt(queryEmbedding.reduce((acc, v) => acc + v * v, 0));
  const normT = Math.sqrt(topicEmbedding.reduce((acc, v) => acc + v * v, 0));

  if (normQ === 0 || normT === 0) {
    return 0;
  }

  const cos = dot / (normQ * normT);
  // Normalize from [-1, 1] to [0, 1]
  return (cos + 1) / 2;
}

/**

 * Compute reputation score from topic metadata.

 * Average of available reputation signals.

 */
export function computeReputationScore(topic: TrendingTopic): number {
  const rep = topic.reputation;
  if (!rep) {
    return 0;
  }

  const components: number[] = [];
  if (typeof rep.avg_author_reputation === "number") {
    components.push(rep.avg_author_reputation);
  }
  if (typeof rep.council_weight === "number") {
    components.push(rep.council_weight);
  }

  if (components.length === 0) {
    return 0;
  }

  return components.reduce((a, b) => a + b, 0) / components.length;
}

/**

 * Get weight configuration for given context.

 * Loads from environment with sensible defaults.

 */
export function getWeights(ctx: SuggestionContext): SuggestionWeights {
  if (ctx === "autocomplete") {
    return {
      semantic: Number(process.env.SUGGESTIONS_AUTOCOMPLETE_WEIGHT_SEMANTIC ?? 0.5),
      trending: Number(process.env.SUGGESTIONS_AUTOCOMPLETE_WEIGHT_TRENDING ?? 0.4),
      reputation: Number(process.env.SUGGESTIONS_AUTOCOMPLETE_WEIGHT_REPUTATION ?? 0.1),
    };
  }

  return {
    semantic: Number(process.env.SUGGESTIONS_RELATED_WEIGHT_SEMANTIC ?? 0.6),
    trending: Number(process.env.SUGGESTIONS_RELATED_WEIGHT_TRENDING ?? 0.2),
    reputation: Number(process.env.SUGGESTIONS_RELATED_WEIGHT_REPUTATION ?? 0.2),
  };
}
```

```typescript
/**

 * Filter topics for autocomplete by prefix match on normalized_text and aliases.

 */
export function filterTopicsForAutocomplete(
  query: string,
  topics: TrendingTopic[]
): TrendingTopic[] {
  const q = query.trim().toLowerCase();

  if (!q) {
    return [];
  }

  return topics.filter((t) => {
    if (t.normalized_text.startsWith(q)) {
      return true;
    }

    return (t.aliases ?? []).some((alias) =>
      alias.toLowerCase().startsWith(q)
    );
  });
}

```

```typescript
/**

 * Rank suggestions based on context, query embedding, and topics.

 * Returns sorted list filtered by min_score and limited to limit.

 */
export function rankSuggestions(
  ctx: SuggestionContext,
  query: string,
  queryEmbedding: number[],
  topics: TrendingTopic[],
  limit: number,
  minScore: number
): RankedSuggestion[] {
  const { RankedSuggestion } = require("./types"); // For type reference
  const weights = getWeights(ctx);

  const candidates = topics.map((topic) => {
    const semanticSim = computeSemanticSimilarity(
      queryEmbedding,
      topic.semantic?.embedding
    );
    const trendingScore = topic.scores.trending ?? 0;
    const reputationScore = computeReputationScore(topic);

    let score =
      weights.semantic * semanticSim +
      weights.trending * trendingScore +
      weights.reputation * reputationScore;

    // Clamp to [0, 1]
    score = Math.max(0, Math.min(1, score));

    // Build reason string
    const reasonParts: string[] = [];
    if (semanticSim >= 0.7) {
      reasonParts.push("High semantic overlap");
    } else if (semanticSim >= 0.4) {
      reasonParts.push("Moderate semantic overlap");
    }
    if (trendingScore >= 0.7) {
      reasonParts.push(
        `Strong trending signal (${Math.round(trendingScore * 100)}%)`
      );
    }
    if (reputationScore >= 0.7) {
      reasonParts.push("High reputation in council-linked threads");
    }

    const reason =
      reasonParts.length > 0
        ? reasonParts.join("; ")
        : "Suggested based on combined semantic, trending, and reputation signals";

    const result: RankedSuggestion = {
      id: topic.id,
      type: "query",
      text: topic.text,
      normalized_text: topic.normalized_text,
      suggestion_source: ctx,
      score,
      semantic_similarity: semanticSim,
      trending_score: trendingScore,
      reputation_score: reputationScore,
      reason,
      shared_context: topic.metadata?.shared_context,
    };

    return result;
  });

  return candidates
    .filter((c) => c.score >= minScore)
    .sort((a, b) => b.score - a.score)
    .slice(0, limit);
}
```

```typescript
// services/noosphere-service/src/suggestions/routes.ts
import express, { Router, Request, Response } from "express";
import { rankSuggestions, filterTopicsForAutocomplete } from "./ranker";
import {
  TrendingTopicsState,
  AutocompleteRequest,
  RelatedRequest,
} from "./types";
import logger from "../../logger"; // Adjust path as needed

type StateLoader = () => Promise<TrendingTopicsState>;

/**

 * Create suggestion routes with given state loader.

 * Assumes state loader returns TrendingTopicsState (topics + metadata).

 */
export function createSuggestionRoutes(stateLoader: StateLoader): Router {
  const router = express.Router();
  let cachedState: TrendingTopicsState | null = null;
  let lastStateLoad: number = 0;
  const CACHE_DURATION_MS = 5 * 60 * 1000; // 5 minutes

  /**

   * Load state from file (with caching).

   */
  async function loadState(): Promise<TrendingTopicsState> {
    const now = Date.now();
    if (cachedState && now - lastStateLoad < CACHE_DURATION_MS) {
      return cachedState;
    }

    cachedState = await stateLoader();
    lastStateLoad = now;
    return cachedState;
  }

  /**

   * Compute embedding for query (mock for now, integrate with Venice AI later).

   */
  function getQueryEmbedding(query: string): number[] {
    // TODO: Integrate with Venice AI API for real embeddings
    // For now, return a mock embedding based on query length
    const embedding = new Array(768).fill(0);
    for (let i = 0; i < query.length; i++) {
      embedding[i % 768] += query.charCodeAt(i) / 1000;
    }
    return embedding;
  }

  /**

   * GET /search/autocomplete

   */
  router.get("/search/autocomplete", async (req: Request, res: Response) => {
    try {
      const { q, limit = 10, session_id } = req.query as AutocompleteRequest & {
        session_id?: string;
      };

      if (!q || typeof q !== "string") {
        return res.status(400).json({ error: "Missing required parameter: q" });
      }

      const state = await loadState();
      const queryEmbedding = getQueryEmbedding(q);

      // Filter by prefix
      const filtered = filterTopicsForAutocomplete(q, state.topics);

      // Rank with autocomplete weights
      const suggestions = rankSuggestions(
        "autocomplete",
        q,
        queryEmbedding,
        filtered,
        Math.min(Number(limit) || 10, 50), // cap at 50
        0.5 // min_score for autocomplete
      );

      logger.debug("autocomplete results", {
        query: q,
        count: suggestions.length,
        session_id,
      });

      return res.json({
        query: q,
        suggestions,
        timestamp: new Date().toISOString(),
      });
    } catch (error) {
      logger.error("autocomplete error", { error });
      return res.status(500).json({ error: "Internal server error" });
    }
  });

  /**

   * GET /search/related

   */
  router.get("/search/related", async (req: Request, res: Response) => {
    try {
      const { query, limit = 10, min_score = 0.5 } = req.query as RelatedRequest &
        {
          min_score?: string;
        };

      if (!query || typeof query !== "string") {
        return res
          .status(400)
          .json({ error: "Missing required parameter: query" });
      }

      const state = await loadState();
      const queryEmbedding = getQueryEmbedding(query);
      const minScore = Math.max(0, Math.min(1, Number(min_score) || 0.5));

      // Rank all topics (no prefix filter for related)
      const suggestions = rankSuggestions(
        "related",
        query,
        queryEmbedding,
        state.topics,
        Math.min(Number(limit) || 10, 50), // cap at 50
        minScore
      );

      logger.debug("related results", {
        query,
        count: suggestions.length,
        min_score: minScore,
      });

      return res.json({
        query,
        related_suggestions: suggestions,
        timestamp: new Date().toISOString(),
      });
    } catch (error) {
      logger.error("related error", { error });
      return res.status(500).json({ error: "Internal server error" });
    }
  });

  return router;
}
```

```typescript
// services/noosphere-service/src/suggestions/trending.ts
import fs from "fs/promises";
import path from "path";
import logger from "../../logger"; // Adjust path as needed
import { TrendingTopicsState } from "./types";

/**

 * Load trending topics state from file.

 * Returns empty state if file not found or invalid.

 */
export async function loadTrendingState(
  filePath: string
): Promise<TrendingTopicsState> {
  try {
    const data = await fs.readFile(filePath, "utf-8");
    const state: TrendingTopicsState = JSON.parse(data);

    // Validate structure
    if (!Array.isArray(state.topics)) {
      throw new Error("Invalid state structure: topics must be array");
    }

    logger.debug("loaded trending state", {
      path: filePath,
      topic_count: state.topics.length,
      last_updated: state.last_updated,
    });

    return state;
  } catch (error) {
    if (error instanceof Error && error.message.includes("ENOENT")) {
      logger.warn("trending state file not found, returning empty state", {
        path: filePath,
      });
    } else {
      logger.error("error loading trending state", { error, path: filePath });
    }

    // Return empty state as fallback
    return {
      version: 1,
      last_updated: new Date().toISOString(),
      time_window_seconds: 3600,
      total_queries: 0,
      topics: [],
    };
  }
}

/**

 * Get default path to trending state file.

 * Resolves to workspace/classical/trending-topics-state.json

 */
export function getTrendingStatePath(): string {
  const workspaceRoot = process.env.MOLTBOT_WORKSPACE_ROOT || "./workspace";
  return path.join(workspaceRoot, "classical", "trending-topics-state.json");
}
```

```markdown
## Search Query Suggestions (P4.3)

Hybrid trending + semantic suggestion system prevents query fatigue by surfacing relevant topics.

### Architecture

**Components:**

- **Trending Detector** (`scripts/trending-topics-consolidation.sh`): Hourly batch job detecting emerging topics from query audit logs via TF-IDF + frequency + recency decay

- **Suggestion Ranker** (`services/noosphere-service/src/suggestions/ranker.ts`): In-memory ranker with environment-driven weight tuning (separate configs for autocomplete vs related)

- **Suggestion API** (`services/noosphere-service/src/suggestions/routes.ts`): Two REST endpoints (`/search/autocomplete`, `/search/related`)

### Data Flow

```
[Query Audit Logs] (JSONL)
    ↓ (hourly cron)
[Trending Detector] (TF-IDF × Frequency × Recency)
    ↓
[trending-topics-state.json] (with embeddings + reputation)
    ↓ (5-min cache)
[In-Memory Ranker]
    ↓ (per-request semantic + scoring)
[/search/autocomplete | /search/related]

```

### Scoring Formula

```
score(topic, query) =
  w_semantic × semantic_similarity(query, topic_embedding) +
  w_trending × trending_score(topic) +
  w_reputation × reputation_score(topic)

```

**Autocomplete Weights** (default: trending-dominant):
- semantic: 0.5

- trending: 0.4

- reputation: 0.1

**Related-Search Weights** (default: semantic-first):
- semantic: 0.6

- trending: 0.2

- reputation: 0.2

**Tuning:** Set `SUGGESTIONS_*_WEIGHT_*` environment variables to adjust.

### Endpoints

- **GET /search/autocomplete?q=...&limit=...**: Prefix-filtered trending-weighted suggestions

- **GET /search/related?query=...&limit=...&min_score=...**: Semantic-first thematic suggestions

See [README.md](#suggestion-endpoints) for examples.

```
